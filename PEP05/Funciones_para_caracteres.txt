cadena.upper(): Devuelve la cadena en mayúsculas.

cadena.lower(): Devuelve la cadena en minúsculas. 'PEDRO'.lower()→ 'pedro'

cadena.islower(): Devuelve True → si todos los caracteres alfabéticos están en minúscula, Devuelve False → en cualquier otro caso
    texto = "hola mundo"
    print(texto.islower())  # True

cadena.capitalize(): Devuelve la cadena con la primera letra en mayúscula.'pedro'.capitalize()→ 'Pedro'

cadena.title(): Devuelve la cadena con la primera letra de cada palabra en mayúscula.'pedro perez'.title()→ 'Pedro Perez'

cadena.istitle():   Devuelve True → si la cadena cumple el formato título, Devuelve False → en cualquier otro caso

    texto = "Organización De Las Naciones Unidas"
    print(texto.istitle())  # True

cadena.strip(): Elimina los espacios en blanco al principio y al final de la cadena. ' hola'.strip()→ 'hola'

cadena.split(separator): Divide la cadena en una lista de subcadenas utilizando elseparador especificado. 'a,b,c'.split(',')→ ['a', 'b', 'c']

cadena.replace(old, new): Sustituye todas las apariciones de la subcadena oldpor new.
'hola'.replace('o', 'a')→ 'hala'

find(sub): Devuelve el índice de la primera aparición de la subcadena sub. Si no la
encuentra, devuelve -1. 'hola'.find('o')→ 0

min(objeto): La función min() encuentra el elemento mínimo de la secuencia pasada
como argumento. Existe una condición, la secuencia (cadena o lista) no puede estar
vacía, de lo contrario obtendrás una excepción ValueError.

max(objeto): La función max() encuentra el elemento máximo de la secuencia
pasada como argumento. Existe una condición, la secuencia (cadena o lista) no puede
estar vacía, de lo contrario obtendrás una excepción ValueError

cadena.isdigit(): Devuelve True si todos los caracteres de la cadena son dígitos. (La cadena no puede estar vacía.)
'1234'.isdigit()→ True

cadena.isnumeric(): es un método de las cadenas (str) que sirve para comprobar si una cadena 
representa un valor numérico, en un sentido más amplio que isdigit(). (La cadena no puede estar vacía.)
'123'.isnumeric()     # True
"-123".isnumeric()   # False (signo menos)
